# PAA2-Kruskal
Escreva um programa em Linguagem C para o algoritmo de Kruskal, que a partir
de um grafo não dirigido, com pesos nas arestas, encontra a árvore geradora de custo
mínimo. Como vimos em classe o algoritmo vai acrescentando arestas, da menos
pesada para a mais pesada, desde que não formem ciclos, até obter a árvore geradora
final. Faz parte da solução encontrar uma boa representação para grafos, bem como
testar se o algoritmo já encontrou a árvore geradora e também como testar se a adição
de uma dada aresta forma um ciclo.
1 Observações
1. Este exercício programa pode ser feito em grupos de até 3 alunos.
2. Apenas um aluno do grupo deve submeter o programa e deve incluir o nome
dos outros membros do grupo no cabeçalho do programa (dentro do arquivo
fonte).
3. A entrada se dará por meio de um arquivo in.txt fornecido pelos alunos que descreve o grafo ponderado de entrada da figura da página 2. A decisão de como
representar o grafo fica por conta dos alunos.
4. Idente e comente seu programa.
